The Basic Language of Web: HTML

Posted by Akinde Samuel on Saturday, November 18th 2023

All morden websites and web applications were built using three fundamental technologies: HTML, CSS and Javascript. These are the languages of the web

In this post, let's focus on HTML. We will learn what HTML is all about, and why you should learn it

What is HTML

HTML stand for Hyper Text Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language)

HTML consist of elements that describes different types of content: paragraphs,links, headings, images, video, etc. Web browsers understands HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDF web docs

Why should you learn HTML?

There are countless reasons for learning the fundamental language of the web. Here are five of them:

  • To be able to use the fundamental web dev language
  • To Hand-draft beautiful websites instead of relying on tools like worpress or wix
  • To build web applications
  • To impress friends
  • To have fun
